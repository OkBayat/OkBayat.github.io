# کتابخانه‌ی منابع راهبری

این بخش مجموعه‌ای از فایل‌های منبع، ترجمه‌ها، اقتباس‌ها و یادداشت‌های مفهومی است که در مطالعه و تسهیل راهبری از آن‌ها استفاده کرده‌ام.

وجود یک متن در این کتابخانه به این معنی نیست که همه‌ی ادعاهای آن، نتیجه‌ی پژوهش شخصی من یا یک نتیجه‌ی علمیِ تأییدشده‌اند. هنگام خواندن هر صفحه باید روشن باشد:

- نویسنده و عنوان اصلی چیست؛
- منبع اصلی کجاست؛
- متن ترجمه‌ی کامل، ترجمه‌ی منتخب، خلاصه یا اقتباس است؛
- کدام بخش از نویسنده‌ی اصلی است؛
- کدام بخش توضیح یا تفسیر من است؛
- چه شواهدی برای ادعای عملی مورد نظر وجود دارد؛
- چه محدودیت یا نقدی همچنان باقی است.

برای بررسی دقیق‌تر، اولویت با صفحه‌ی اصلیِ نویسندگان است. اگر نسخه‌ی اصلی به‌روزرسانی می‌شود، به‌جای بازنشر آن در سایت‌های دیگر، به منبع اصلی ارجاع دهید.
