# Book Notes

Book Notes document what I am learning from a specific book: its main ideas, questions it raises, points of disagreement, and connections to my work.

These pages should distinguish among:

- the author's argument;
- direct quotation;
- my interpretation;
- practical implications;
- questions that require further evidence.

A Book Note is not a substitute for the original work, and an interpretation should not be presented as the author's own conclusion.
