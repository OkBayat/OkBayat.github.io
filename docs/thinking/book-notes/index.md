# Book Notes Archive

This is the legacy book-specific archive. It remains available so existing URLs and child-page relationships are not broken.

The current user-facing section is [Reading Notes](/thinking/reading-notes), which includes notes developed from books, papers, talks, and other identifiable sources.

Book-specific pages in this archive should continue to distinguish among:

- the author's argument;
- direct quotation;
- my interpretation;
- practical implications;
- questions that require further evidence.

A reading note is not a substitute for the original work, and an interpretation should not be presented as the source author's own conclusion.
